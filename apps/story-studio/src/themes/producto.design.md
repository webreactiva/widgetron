# Producto — the neutral second theme

A calm teal/amber identity, deliberately distant from Web Reactiva's
terracotta/lilac/mustard, used to PROVE the theme pipeline: same guide, one
different `design.md`, whole new brand. Rounded corners and soft shadows where
Web Reactiva is sharp and solid — the contrast makes theme bugs obvious.
